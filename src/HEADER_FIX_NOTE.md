# ملاحظة مهمة: إصلاح خطأ في Header.tsx

## المشكلة
يوجد خطأ بسيط في الملف `/components/layout/Header.tsx` في السطر 340.

## الخطأ
السطر 336 يبدأ بـ `<button>` لكن السطر 340 ينتهي بـ `</a>` بدلاً من `</button>`

## موقع الخطأ
```
السطر 336: <button key={idx} onClick={() => { onNavigate('riyadh-rent'); setIsWasataOpen(false); }} ...>
...
السطر 340:   </a>    ❌ يجب أن يكون </button>
```

## الحل
قم بتغيير السطر 340 من:
```tsx
                                  </a>
```

إلى:
```tsx
                                  </button>
```

## كيفية الإصلاح يدوياً
1. افتح الملف `/components/layout/Header.tsx`
2. اذهب إلى السطر 340
3. قم بتغيير `</a>` إلى `</button>`
4. احفظ الملف

## ملاحظة
حاولت الأدوات الآلية إصلاح هذا الخطأ لكن فشلت بسبب مشاكل في ترميز الأحرف العربية في الملف. الحل الأفضل هو التعديل اليدوي البسيط.
